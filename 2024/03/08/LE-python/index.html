python环境的安装（“学习python的第0.5步”）

  1. 点击https://repo.anaconda.com/miniconda/Miniconda3-latest-Windows-x86_64.exe
    进行安装，请务必一定千万要记住你的miniconda3下在了哪里

  2. 四个全选

  3. 或许是测试（？）总之我也不大清楚，但是jhgg这样做总有他的深意吧

    1. 按下Win键，输入wt

    2. 输入conda info -e

    3. 输入conda activate base

    4. 输入python

    5. 输入print("Hello, world!")

pycharm的安装

  1. https://www.jetbrains.com/pycharm/download/?section=windows下载

  2. next…next…直到…

    可以勾上“添加‘bin’文件夹到PATH”（PATH那个选项不要选。如果装过其他版本的python，它也会被添加到path里，其他软件执行python解释器和虚拟环境可能会错误调用。所以先取消勾选，然后再手动一下配置环境变量），可以考虑勾上“创建桌面快捷方式”，不要勾上另外两个（不要问我为什么，我也不知道，诶嘿）

  3. 汉化：打开pycharm，左侧有"plugins"，点击打开，在插件商店搜索"chinese"

    下载后重启pycharm

  4. 还有一些要做的…

    1. 新建文件（呃还是新建项目来着？总之是打开pycharm后右侧大块区域里靠左边的那个框框点击去）

    2. 自定义环境

    3. 选择现有

    4. 类型->conda

    5. 点击这里找找你装conda脚本的位置，一般是：C:\Users\…\Miniconda3\condabin\conda.bat

    6. 创建

  5. 大功告成！
