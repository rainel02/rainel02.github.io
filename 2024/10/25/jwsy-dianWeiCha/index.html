【学习经验】电位差预习报告

1 自组电位差计实验

1.1 实验器材

直流稳压电源

  • 有两路输出，TRACKING键都弹出时各路输出可单独使用
  • 按下power键打开电源开关，面板显示额定输出电压和额定输出电流，可使用电压/电流调节旋钮调节。外电路连接完毕检查无误后按下output键，此时电源对外电路才有输出。
  • 注意，在检查线路故障或拆线前先要关闭output，实验结束关闭电源开关。

标准电池和电阻箱


指针式检流计

  • 使用时将制动旋钮逆时针旋转到白色标记处解锁指针，调零。
  • 检测电路中是否有电流通过：按下电计键。
    • 电路中电流较弱，指针不会满偏时：顺时针旋转锁定电计键。
    • 使指针快速稳定下来：按下短路键
  • 使用完毕：弹起电计键，制动旋钮放在红锁标记处以锁定指针（为了保护检流计的指针）

点触式大电阻

  • 开关弹起，串联 $30k\Omega$
  • 开关按下，$30k\Omega$ 电阻短路

1.2 实验设计

1.3 实验步骤

自组电位差计测干电池电动势

  1. 调节工作电流

    1. 计算 $E_{N} \approx E_{20}-3.99\times10^{-5}(t-20℃)-0.94\times10^{-6}(t-20℃)^2+9\times10^{-9}(t-20)^3$
    2. 计算并设置 $R_{1}=\frac{E_{N}}{I_{0}}$
    3. 计算并设置 $R_{2} \approx \frac{E-E_{N}}{I_{0}}$ ，
    4. 接通辅助回路和第一补偿回路，使用带有大电阻的点触式开关，调节 $R_{2}$ ，使检流计无偏转（按下点触式开关，按下电计键，若检流计向右偏转，$R_2$ 调大；若检流计向左偏转，$R_2$ 调小）
    5. 检流计不偏转时，记录 $R_{1}, R_{2}$ ，有表达式 $E_{N} = R_{1}I_{0} = \frac{R_{1}}{R_{1}+R_{2}}E\ (1)$

  2. 测量待测电压

    1. 接通辅助回路和第二补偿回路
    2. 保持$I_{0} = 1mA$ 不变，同时调节 $R_{1},R_{2}$ 使检流计不偏转（按下点触式开关，按下电计键，若检流计向右偏转，$R_2$ 调大 $R_1$ 调小；若检流计向左偏转，$R_2$ 调小 $R_1$ 调大）
    3. 记录 $R_{1}‘,R_{2}’$，有表达式 $E_{X}=R_{1}‘I_{0}= \frac{R_{1}’}{R_{1}‘+R_{2}’}E\ (2)$
    4. 由表达式 $(1)(2)$ 计算 $E_{X} = \frac{E_{N}(R_{1}+R_{2})}{R_{1}(R_{1}‘+R_{2}’)}R_{1}'$

测量自组电位差计的灵敏度

  1. 保持$I_{0} = 1mA$ 不变，同时调节 $R_{1},R_{2}$ 使检流计偏转 $10-14div$（按下点触式开关，按下电计键，若检流计向右偏转，$R_2$ 调大 $R_1$ 调小；若检流计向左偏转，$R_2$ 调小 $R_1$ 调大）
  2. 记录电阻箱1和2的阻值分别为 $R_{1}‘’, R_{2}‘’$，
  3. 计算灵敏度 $S = \frac{检流计偏转格数}{I_{0}|R_{1}‘-R_{2}’|}(div/V)$
  4. 计算灵敏度误差 $\Delta_{灵}(E_{X}) = \frac{0.2div}{S}(V)$

2 箱式电位差计实验

2.1 实验器材

UJ25型箱式电位差计

测量上限：1.911110V
准确度等级：0.01级
工作电流：$I_{0}= 0.1mA$

  • $R_{AB}$：修正不同温度下标准电池 $E_N$ 的小数点后第四位和第五位的数值

  • $R_{P}$：调节工作电流 $I_0$

  • $I_{0}R_{CD}$：测量未知电压的值

  • 左侧的功能转换开关

    • 断：电位差计不工作
    • 标准：接入标准电池 $E_N$
    • 未知1/未知2：测未知1或未知2的位置电压

  • 左下角的按钮：检流计的控制开关

    • 通常断开
    • 粗：接入电路，串联一大电阻，用以在远离补偿的情况下，保护检流计
    • 细：检流计直接接入电路
    • 短路：阻尼开关，使线圈迅速停止摆动

电子检流计

  • 不使用时功能档转换旋钮转至表头保护位置
  • 使用时先打开背板开关，在无电流输入检流计的情况下把功能挡转换旋钮调至调零位置，用调零旋钮调节指针为0
  • 功能挡转换旋钮调至补偿位置，用补偿旋钮进一步指针调0
  • 测量时，如果检流计待测电流大小未知，可使用非线性档粗调，此时量程无穷大，将待测电流调整到0点附近再进行测量。
  • 顺时针旋转功能档，检流计量程由1微安到3纳安逐步变小
  • 实验完毕，功能档归位到表头保护位置，并关闭电源

2.2 实验步骤

  1. 检流计机械调零
  2. 计算 $E_N$ ，功能转换开关旋至“标准”档，调节 $R_{AB}$ 修正 $E_{N}$ 的后两位
  3. 按左下角按钮“粗”“细”，调节 $R_{P}$ 的粗中细微四个旋钮至检流计指0
  4. 功能转换开关旋至“未知1”或“未知2”，分别按左下角按钮“粗”“细”，调节 $R_{CD}$ 至检流计指0，示值即为待测电动势 $E_X$

3（选做）箱式电位差计测电阻

实验步骤

  1. 接线路： $R_N$ 接“未知1”，$R_X$ 接“未知2”
  2. 检流计机械调零
  3. 计算 $E_N$ ，功能转换开关旋至“标准”档，调节 $R_{AB}$ 修正 $E_{N}$ 的后两位
  4. 按左下角按钮“粗”“细”，调节 $R_{P}$ 的粗中细微四个旋钮至检流计指0
  5. 功能转换开关旋至“未知1”，分别按左下角按钮“粗”“细”，调节 $R_{CD}$ 至检流计指0，示值即为 $R_N$
  6. 功能转换开关旋至“未知2”，分别按左下角按钮“粗”“细”，调节 $R_{CD}$ 至检流计指0，示值即为 $R_X$
